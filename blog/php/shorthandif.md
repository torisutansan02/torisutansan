# Shorthand If

You can write if statements in one line.

```html
<!DOCTYPE html>
<html>

<body>
<?php
    $x = 10;
    if ($x == 10) echo "Nice!";
?>
</body>

</html>
```

## Shorthand If-Else

Can also be written in one line.
- Ternary operatory / conditional expression.

```html
<!DOCTYPE html>
<html>

<body>
<?php
    $x = 5;
    if ($x == 4) ? "Nice!" : "Not nice!";
?>
</body>

</html>
```
